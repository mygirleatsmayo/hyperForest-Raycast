# HyperForest Raycast Theme

A custom dark, vibrant forest theme.

## Installation

Click the link below to install directly into Raycast:

[**Install HyperForest**](https://themes.ray.so?version=1&name=hyperForest&author=mygirleatsmayo&authorUsername=mygirleatsmayo&colors=%23001700,%23001700,%23FFB3D9,%2372FF72,%23FF1493,%23FF3DB8,%23FF69B4,%23F1C40F,%232ECC71,%235C59B6,%23BB8FCE,%23FF1493&appearance=dark&addToRaycast)

Alternatively, you can import the `hyperForest.json` file if supported, or manually copy the hex values.
