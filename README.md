# HyperForest Raycast Theme

A dark, vibrant forest theme for Raycast, inspired by VS Code HyperForest.

## Installation

1. Open [Raycast Theme Studio](https://themes.ray.so/)
2. Drag and drop `hyperForest.json`
3. Click "Add to Raycast"

Or use this direct link:
